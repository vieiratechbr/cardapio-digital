# 🔥 Brasa & Craft — Cardápio Digital

> **⚠️ AVISO IMPORTANTE:** Este projeto é um **modelo de demonstração** criado exclusivamente como portfólio. O restaurante "Brasa & Craft", seus produtos, preços, avaliações e informações de contato são **totalmente fictícios**. Nenhum pedido real pode ser realizado através desta página.

---

## 📋 Sobre o Projeto

Este é um **cardápio digital profissional** desenvolvido como mostruário comercial para demonstrar a capacidade de criação de soluções digitais para restaurantes, lanchonetes, hamburguerias e negócios alimentícios em geral.

O objetivo é apresentar a potenciais clientes como um cardápio digital moderno, funcional e focado em conversão pode transformar a experiência de pedidos do seu estabelecimento.

---

## 🎯 O que este modelo demonstra

- Design profissional, moderno e focado em conversão
- Integração direta com **WhatsApp** para recebimento de pedidos
- **Carrinho de compras** funcional sem necessidade de backend
- Geração automática de **mensagem formatada** para o WhatsApp
- Experiência otimizada para **dispositivos móveis** (mobile-first)
- Navegação fluida entre categorias do cardápio
- Animações e micro-interações profissionais

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| **HTML5** | Estrutura semântica da página |
| **CSS3** | Estilização, animações e responsividade |
| **JavaScript (Vanilla)** | Lógica do carrinho, FAQ e interações |
| **Google Fonts** | Bebas Neue (display) + DM Sans (corpo) |

Sem frameworks, sem dependências externas complexas. Código limpo, leve e fácil de personalizar.

## ✨ Funcionalidades Implementadas

- ✅ Header sticky com logo, horário e localização
- ✅ Hero section com hambúrguer animado em CSS puro
- ✅ Barra de aviso de taxa de entrega e pedido mínimo
- ✅ Navegação por categorias com scroll suave e destaque ativo
- ✅ Cards de produtos com badges (Mais Pedido, Novo, Promoção, Artesanal)

- ✅ Toast notifications ao adicionar itens
- ✅ Seção de diferenciais do estabelecimento
- ✅ Avaliações de clientes com estrelas
- ✅ FAQ com accordion interativo
- ✅ Botão flutuante do WhatsApp com animação de pulso
- ✅ Scroll reveal animado nos cards
- ✅ 100% responsivo e mobile-first

## 🚀 Como Visualizar

1. Faça o download ou clone este repositório
2. Coloque os 3 arquivos (`index.html`, `style.css`, `script.js`) na mesma pasta
3. Abra o arquivo `index.html` em qualquer navegador moderno
4. Nenhuma instalação ou servidor é necessário

---

## 🔧 Como Personalizar para um Cliente Real

Para adaptar este modelo a um estabelecimento real, basta editar:

| O que mudar | Onde fica |
|---|---|
| Nome e slogan do restaurante | `index.html` — seção header e hero |
| Número do WhatsApp | `index.html` — todos os links `wa.me/` e `script.js` (variável `numero`) |
| Produtos, preços e descrições | `index.html` — seção de cada categoria |
| Cores e identidade visual | `style.css` — variáveis CSS no `:root` |
| Endereço e horários | `index.html` — seções header e footer |
| Taxa de entrega e pedido mínimo | `index.html` (delivery-bar) e `script.js` (finalizarPedido) |
